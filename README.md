# FinTech Wallet App

A full-stack digital wallet application with authentication, secure payments,
transaction management, and complete DevSecOps pipeline implementation.

## Features (initial)
- User authentication (JWT + refresh tokens)
- Role-based dashboard
- Add money, send money, withdraw features
- Payment gateway integration (Stripe)
- Dockerized backend & frontend
- Complete CI/CD pipeline (GitHub Actions)
- Security scanning & automated tests
